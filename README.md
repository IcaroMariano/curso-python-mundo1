Curso de Python 3 – Mundo 1 (Curso em Vídeo)

Este repositório contém **todos os exercícios resolvidos do Mundo 1 – Fundamentos**, do curso ministrado por **Gustavo Guanabara** no canal **Curso em Vídeo**.

O objetivo deste repositório é registrar meu aprendizado em Python, praticar lógica de programação e criar um histórico público da minha evolução na linguagem.

---

## Conteúdos abordados

Durante o Mundo 1, foram estudados os seguintes conceitos:

* Instalação e configuração do Python
* Variáveis e tipos primitivos
* Entrada e saída de dados
* Operadores aritméticos e lógicos
* Estruturas condicionais (`if`, `elif`, `else`)
* Laços de repetição (`for`)
* Manipulação básica de strings
* Lógica de programação aplicada
